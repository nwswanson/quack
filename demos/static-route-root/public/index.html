Quack route-root demo

Static routes can mount different archive folders at public URL paths.

This page is served by path: / with root: public. The stylesheet is not inside public/; it is mounted from media/css through the /css route.

Routes Under Test

URL route Archive root Example link
/ public Home index
/css media/css Mounted stylesheet
/docs content/docs Docs index
/shared alternate/shared Shared about page

site.yml

This is the manifest used for this demo upload.

routes:
  - path: /
    kind: static
    root: public
  - path: /css
    kind: static
    root: media/css
  - path: /docs
    kind: static
    root: content/docs
  - path: /shared
    kind: static
    root: alternate/shared

Path Shadowing Checks

Longer matching static routes win. These links prove that files under public/ do not accidentally win when a more specific route exists.

/docs/shadow.html uses content/docs /shared/about.html uses alternate/shared /css/main.css uses media/css /css/ directory index
